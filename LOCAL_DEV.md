# Running the Web App Locally

The app is an ASP.NET Core project located at `WebAppWithDB_Starter_v2/WebAppWithDB_Starter_v2/`.

## Start the server

```bash
cd WebAppWithDB_Starter_v2/WebAppWithDB_Starter_v2
dotnet run
```

Once started, open **http://localhost:5005** in your browser.

## Stop the server

Press `Ctrl+C` in the terminal where `dotnet run` is running.

## Prerequisites

- [.NET SDK](https://dotnet.microsoft.com/download) installed (`dotnet --version` to verify)
